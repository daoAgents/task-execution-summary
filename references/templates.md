## 模板变量参考

### 元数据变量

| 变量名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `task_name` | string | - | 任务名称 |
| `generated_at` | datetime | 当前时间 | 报告生成时间 |
| `detail_level` | enum | "standard" | 详细程度 |
| `language` | string | "zh-CN" | 报告语言 |

### 任务信息变量

| 变量名 | 类型 | 说明 |
|--------|------|------|
| `task_scope` | string | 任务范围描述 |
| `start_time` | datetime | 任务开始时间 |
| `end_time` | datetime | 任务结束时间 |
| `total_duration` | string | 总耗时（格式化） |
| `participant_count` | number | 参与人数 |

### 分析结果变量

| 变量名 | 类型 | 说明 |
|--------|------|------|
| `achievement_rate` | number | 目标达成率 (0-100) |
| `total_issues` | number | 问题总数 |
| `quality_score` | number | 综合质量评分 (0-100) |
| `completeness_rate` | number | 信息完整度 (0-100) |

### 条件渲染变量

| 变量名 | 类型 | 说明 |
|--------|------|------|
| `has_collaboration` | boolean | 是否包含协作信息 |
| `has_risk_warnings` | boolean | 是否包含风险预警 |
| `is_degraded` | boolean | 是否为降级生成 |

## 模板变体对比

| 章节 | 摘要版 | 标准版 | 详细版 | 学习版 |
|------|--------|--------|--------|--------|
| 第1章 执行概览 | 完整 | 完整 | 完整+扩展数据 | 学习概况 |
| 第2章 背景与目标 | 标题+要点 | 完整 | 完整+约束详述 | 完整+知识图谱 |
| 第3章 执行过程 | 标题+关键节点 | 完整 | 完整+操作级细节 | 学习历程记录 |
| 第4章 关键决策 | 省略 | 完整 | 完整+决策矩阵 | 简化（技术选型） |
| 第5章 问题与方案 | 标题+要点 | 完整 | 完整+根因分析链 | 完整+认知障碍 |
| 第6章 资源使用 | 省略 | 完整 | 完整+利用率评估 | 简化（学习资源） |
| 第7章 协作分析 | 省略 | 条件包含 | 条件包含+详细评估 | 学习支持系统 |
| 第8章 多维分析 | 标题+雷达图 | 完整 | 完整+交叉分析 | 知识掌握度评估 |
| 第9章 经验方法论 | 摘要 | 完整 | 完整+SOP+案例库 | 学习方法论 |
| 第10章 改进建议 | 摘要 | 完整 | 完整+行动计划+甘特图 | 后续学习计划 |
| 附录 | 无 | 无 | 完整4个附录 | 学习产出物 |
| **预计篇幅** | **500-800字** | **3000-5000字** | **8000-15000字** | **3000-6000字** |

| 你的需求 | 推荐模板 | 理由 |

| 项目结项总结 | **标准版** | 结构完整，适合归档 |
| 重大故障事后分析 | **详细深度版** | 需要完整的证据链和追溯能力 |
| 外部审计/合规要求 | **详细深度版** | 信息完整，经得起审查 |
| 作为培训案例材料 | **详细深度版** | 细节丰富，可供深入研究 |
| 个人学习总结 | **学习专用版** | 聚焦知识获取和能力成长 |

| 导师制学习档案 | **学习专用版** | 包含导师指导和成长轨迹 |

### 选择决策流程

```
开始选择模板
    │
    ├── 这是学习类任务吗？
    │   ├── 是 → 使用【学习专用模板】
    │   └── 否 ↓
    │
    ├── 需要多详细的报告？
    │   ├── 只需要快速了解概要 → 使用【快速摘要版】
    │   ├── 需要标准的完整报告 → 使用【标准版】（默认推荐）
    │   └── 需要极度详尽（审计/案例研究） → 使用【详细深度版】
    │
    └── 不确定？ → 从【标准版】开始，可根据需要扩展或精简
```

### 模板间转换说明

所有模板变体基于相同的底层信息模型，可在不同模板间转换：

- **摘要版 → 标准版**：补充第2-9章详细内容
- **标准版 → 详细版**：每章增加更多细节数据，添加附录A-E
- **标准版 → 学习版**：调整各章节角度（从"完成任务"转为"学习知识"），替换第七章为学习支持系统
- **任意版本 → 摘要版**：提取第1章和第10章核心内容，精简呈现
